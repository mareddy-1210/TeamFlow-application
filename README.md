# TeamFlow Application

## Project Overview
TeamFlow is a Project Management System developed using Spring Boot. It provides REST APIs for managing users, projects, tasks, comments, notifications, reviews, RCA (Root Cause Analysis), and task relationships.

## Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

## Features
- User Management
- Project Management
- Task Management
- Comments
- Notifications
- Reviews
- Root Cause Analysis
- Task Dependencies

## API Endpoints
- /users
- /projects
- /tasks
- /comments
- /notifications
- /reviews
- /rcas
- /taskrelations

## How to Run
1. Clone the repository.
2. Configure PostgreSQL.
3. Update application.properties.
4. Run:

```
.\mvnw spring-boot:run
```
